# Whisky Monitor

Автоматический мониторинг новых поступлений виски с европейских магазинов.

## Отслеживаемые магазины

- WhiskyAgents
- Whiskyfass
- WhiskySite
- Maltucky
- CaptainScotch
- DeinWhisky

## Возможности

- Проверка новых поступлений каждые 15 минут
- Уведомления в Telegram
- Сохранение уже найденных бутылок
- Автоматическое обновление базы известных товаров
